# Scouting the Academy's Next Star

A NumPy mini-project for analyzing player performance and selecting a final academy squad based on weighted talent scores.

## Project Overview

This project simulates a sports academy scouting process. Player performance data is analyzed using NumPy to calculate talent scores, rank players, and select a final six-player squad.

A selection rule ensures that the final squad contains at least one goalkeeper.

## Skills Practiced

- NumPy arrays
- Array indexing and slicing
- Array copying
- Data cleaning with `np.clip`
- Broadcasting
- Weighted calculations
- Aggregation with `axis`
- Structured arrays
- Record arrays
- Boolean filtering
- Sorting
- Squad selection logic
