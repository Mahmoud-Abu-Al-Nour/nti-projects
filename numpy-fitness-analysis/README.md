## Fitness Data Insights

The average number of steps is approximately 7,786 steps per day. The maximum number of steps is 12,000, while the minimum number of steps is 5,000, giving a step range of 7,000.

The data shows that days with a higher number of steps generally have higher calories burned. For example, the day with 12,000 steps also has the highest calorie burn of 400 calories.

Day 7 appears unusual because it has the highest number of steps and calories burned but the lowest amount of sleep at only 5 hours.

The standard deviation of calories is approximately 62.73, which indicates that calorie burn varies across the seven days.

From this data, I learned how NumPy matrices can be used to slice, reshape, filter, and statistically analyze fitness data. I also noticed a possible positive relationship between physical activity and calories burned.
